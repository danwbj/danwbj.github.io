爬楼梯问题（斐波拉契数列）

问题：楼梯总共有 N 级，若每次只能跨上一级或二级或者三级，要走上第 N 级，共有多少种走法？
思路：
N = 1 的时候只有 1 种方法。
N = 2 的时候有 2 方法。
N = 3 的时候有 4 方法。
N = K 的时候，可以先走到 k-1 那里，然后向前走一阶，或者先走到 k-2 那里然后前走两阶
所以后一个答案是前两个的答案之和。
