素数：除了1和它本身外没有其他因数，即，除了1和它本身外不能被其他数整除
1---不是素数
2---最小的素数
3---2（判断是否能被2整除）
4---2，3（判断是否能被2,3整除）
5---2，3，4（判断是否能被2,3,4整除）
6---2，3，4，5（判断是否能被2,3,4,5整除）
...

以下是js代码的实现：
```javascript
for(let i=2;i<=100;i++){
    let flag = true;
    for(let j=2;j<i;j++){
        if(i%j==0){
            flag = false;
            break;
        }
    }
    if(flag){
        console.log(i)
    }  
}
```
